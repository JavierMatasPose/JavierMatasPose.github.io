**Full-stack web application & Reinforcement Learning**

DQNTicTacToe is a web app where you can challenge a self-taught Deep Q-Network (DQN) agent in a game of Tic Tac Toe.

*   **The Agent:** Learned the basics via *self-play* and continues to evolve.
*   **Tech Stack:** Python, Flask (backend), Gymnasium (game mechanics), PyTorch (DL).
